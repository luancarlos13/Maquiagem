## 💄 Landing Page - Maquiagem

## 📌 Sobre o Projeto

Esta landing page foi desenvolvida com o objetivo de apresentar serviços de maquiagem de forma elegante, moderna e altamente atrativa, valorizando a beleza e a experiência do cliente.

O layout foi planejado para destacar os principais serviços, estilos e diferenciais, conduzindo o usuário até ações importantes como agendamento ou contato. Esse tipo de página é focado em conversão, ou seja, transformar visitantes em clientes interessados

## 🎯 Objetivo

O principal objetivo do projeto é simular um site profissional para maquiadoras ou estúdios de beleza, transmitindo:

- 💎 Sofisticação
- 💖 Cuidado com o cliente
- ✨ Qualidade nos serviços

A página busca gerar conexão emocional com o público e incentivar o agendamento de serviços.

## 🛠️ Tecnologias Utilizadas
- HTML5 🟧
- CSS3 🟦
- JavaScript 🟨

## 🧩 Funcionalidades

- 💄 Apresentação de serviços de maquiagem (social, eventos, noivas, etc.)
- 📸 Destaque visual para resultados e estilos
- 📱 Layout totalmente responsivo
- 🎨 Design moderno e feminino
- 📞 Botões de chamada para ação (CTA) para contato/agendamento
- ⭐ Possível inclusão de depoimentos para gerar confiança
- 🎨 Design e Experiência

O design segue uma proposta elegante e minimalista, com foco total na estética — algo essencial no segmento de beleza.

Elementos visuais bem trabalhados ajudam a transmitir profissionalismo, já que páginas de maquiagem dependem fortemente da apresentação visual para gerar credibilidade

A navegação é simples e intuitiva, garantindo uma experiência agradável para o usuário.

## 🚀 Conclusão

Este projeto demonstra a criação de uma landing page voltada para o nicho de beleza, aplicando conceitos importantes como:

Experiência do usuário (UX)
Design visual estratégico
Estrutura focada em conversão

Além disso, serve como base para projetos mais avançados, como agendamentos online ou plataformas completas para profissionais da beleza.
